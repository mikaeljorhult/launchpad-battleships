# Launchpad Battleships
Small game to showcase the [MIDI Events module](https://github.com/mikaeljorhult/midi-events).

Game requires a Novation Launchpad MIDI controller and is only playable in browsers that support
the [Web MIDI API](http://www.w3.org/TR/webmidi/), ie. latest version of Chrome and Chrome Canary
with the flag for experimental support([chrome://flags/#enable-web-midi](chrome://flags/#enable-web-midi))
set to active.
